# 📬 InboxOS Frontend — React Client Dashboard

This is the frontend dashboard client for **InboxOS — The AI Inbox Operating System**.

## Stack
- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State & Queries**: TanStack React Query
- **Routing**: React Router DOM

## Development Setup

1. Copy `.env.example` to `.env` and configure your API URL.
2. Run `npm install` from the workspace root.
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
